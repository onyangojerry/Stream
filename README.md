# Stream

Modern, feature-rich video communication built with React, TypeScript, WebRTC, and Vite. Includes group calls, screen share, recording, transcription, collaborative docs, and a smart waiting room.

## Highlights
- Fast dev experience (Vite), responsive UI (Tailwind), dark/light mode
- WebRTC peer-to-peer with optional auth and role-based controls
- Modular React components and lightweight state (Zustand)

## Quick Start
```bash
git clone https://github.com/onyangojerry/Stream.git
cd Stream
npm install
npm run dev
```
Open http://localhost:3000

## Scripts
- dev: start Vite dev server
- build: production build
- preview: preview built app
- lint: run ESLint
