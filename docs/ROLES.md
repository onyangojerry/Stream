# Roles and Permissions

## Guest
Can:
- view public community surfaces
- submit join requests with required metadata
- RSVP to public scheduled meetings

Cannot:
- start calls
- schedule calls
- publish materials
- react to materials
- post threads or replies
- invite collaborators or create communities

## Authenticated User
Can:
- start and schedule calls
- participate in community contributions
- manage own profile and GitHub metadata
- follow other users
- RSVP and request access to public meetings

## Meeting Host
Can:
- start/end hosted session
- moderate attendee/join-request workflow
- manage in-call tools and call controls

## Admin Context
Admin behavior depends on configured moderation surfaces and policy decisions; this repository currently exposes host-centric moderation patterns in call/community UIs.
